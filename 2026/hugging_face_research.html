How to Use: The pipeline()

The `transformers` library abstracts away the complexity. Select a task below to generate the code.

main.py 

from transformers import pipeline

# 1. Initialize the pipeline for text generation
generator = pipeline("text-generation")

# 2. Run inference with a prompt
result = generator("In the future, AI will be")

print(result)
Downloads the default GPT-2 model and generates text based on your prompt.
